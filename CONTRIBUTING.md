# Contributing to SpatialStratifiedDE

Thank you for your interest in contributing to **SpatialStratifiedDE**! This project was born during the Bioconductor Venice Hackathon 2026, and we welcome improvements, bug fixes, and new features.

## How to Contribute

1.  **Report Bugs:** Use the GitHub Issues to report any bugs or issues you encounter.
2.  **Suggest Features:** If you have an idea for a new feature, please open an issue to discuss it.
3.  **Pull Requests:** 
    -   Fork the repository.
    -   Create a new branch for your feature or fix.
    -   Ensure your code follows the existing style (see `GEMINI.md`).
    -   Submit a pull request with a clear description of your changes.

## Development Standards

-   Follow the [tidyverse style guide](https://style.tidyverse.org/).
-   Ensure all new functions are documented using `roxygen2`.
-   If you add a new feature, please update the vignette or add a new example script.

## Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
